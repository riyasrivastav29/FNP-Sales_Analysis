# FNP-Sales_Analysis
1. Analyzed 1,000 orders generating ₹35.2L revenue using an interactive Excel dashboard.</br>
Built KPIs for total orders, revenue, avg. customer spend (₹3,520), and delivery time (5.53 days).
Used Power Query for data cleaning and transformation.
Applied Pivot Tables & DAX to analyze sales by occasion, category, product, month, hour, and city.
Designed dynamic charts and slicers for real-time filtering and business insights.
