# FNP Sales Analysis -This Dashboard built using Excel, Power Query, Pivot Tables, and DAX to analyze revenue, order trends, delivery timelines, customer spend, top products, occasions, and cities through interactive visualizations and slicers.</br>
1. Analyzed 1,000 orders generating ₹35.2L revenue using an interactive Excel dashboard.</br>
2. Built KPIs for total orders, revenue, avg. customer spend (₹3,520), and delivery time (5.53 days).</br>
3. Used Power Query for data cleaning and transformation.</br>
4. Applied Pivot Tables & DAX to analyze sales by occasion, category, product, month, hour, and city.</br>
5. Designed dynamic charts and slicers for real-time filtering and business insights.</br>
